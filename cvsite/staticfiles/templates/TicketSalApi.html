ETKİNLİK TAKAS PLATFORMU

Önceki TicketSale Web sitem için JAVA Spring ile geliştirdiğim bir RESTful API servisidir. Kullanıcı ve bilet yönetimi işlemlerini destekleyen bu backend uygulaması, katmanlı mimari prensiplerine uygun olarak yapılandırılmıştır. Projede Spring Boot, JPA, Hibernate ve Lombok gibi modern Java teknolojileri kullanılmıştır. Proje içeriğine CONTACT bölümündeki github linkinden ulaşabilirsiniz.
